# 慢病用药小管家

面向慢性病患者及家属的全链路用药管理工具，覆盖「问诊→购药→用药→续方」完整闭环。

## 在线预览

**https://xiaxxue.github.io/med-manager/**

手机浏览器直接打开即可体验完整交互，也可用电脑浏览器访问（建议 Chrome 开发者工具切换到 iPhone 12/13 尺寸）。

## 产品亮点

1. **双角色系统** — 启动页选择"我是患者"或"我是家属"，进入完全不同的界面和功能
2. **AI 智能体助手（小团）** — 不是简单问答，而是 Agent 架构：规划层理解意图 → 调度中心调用工具 → 记忆模块保持上下文 → 输出层生成动作卡片，可直接操作（续方、预约、查库存）
3. **全链路闭环** — 问诊预约 → 进入问诊对话 → 医嘱同步 → 购药下单 → 库存自动更新 → 用药打卡 → 库存预警 → 智能续方，每个环节无缝衔接
4. **家属协同管理** — 独立的 5 页面模式（监护首页/药量/小团助手/问诊/设置），远程查看服药状态、帮续方、帮约医生、家属代问诊（独立界面，医生知晓是家属代为沟通）
5. **智能库存预测** — 根据日均用量自动计算剩余天数，提前 7 天触发续方提醒，防止断药和重复购药
6. **用药时间轴动态化** — 根据真实时间自动切换状态（已过去/当前/未到），支持日期切换查看历史和未来

## 功能一览

### 患者模式（5个Tab）

| 页面 | 核心功能 |
|------|---------|
| 首页 | 顶栏自动收起、AI 健康建议、风险提醒、动态时间轴（打卡/补录/撤回）、最近问诊 |
| 库存 | 4 种药品库存可视化、修正数量、重复购药提醒、购药下单（选药→配送→支付）|
| 小团助手 | Agent 架构、5 大能力模块、思考过程动画、动作卡片、多轮对话 |
| 看医生 | 问诊方式选择、医生列表、预约流程、问诊对话界面、问诊小结同步、续方进度（动态推进）、准备清单（含血压上传）、购药下单（库存自动更新）|
| 个人中心 | 依从性报告、家属管理（添加/绑定/通知）、提醒偏好、用药数据导出、协同管理入口 |

### 家属模式（5个Tab）

| 页面 | 核心功能 |
|------|---------|
| 监护首页 | 服药状态圆点进度、异常提醒、只读时间轴（带提醒按钮）、快捷操作、本周统计 |
| 药量 | 查看爸爸的药品库存、帮续方购药 |
| 小团助手 | 家属视角 Agent（"爸爸今天吃药了吗"、"帮爸爸约医生"）|
| 问诊 | 代问诊说明、帮爸爸预约问诊、独立代问诊对话界面（话术区分）、查看问诊记录 |
| 设置 | 提醒偏好、权限管理、切换角色 |

### 弹窗交互（9个）

依从性报告、跳过原因、添加家属、库存修正、用药数据（统计+图表+导出）、预约问诊、问诊对话、问诊小结、购药下单

## 技术说明

- 纯前端原型，HTML + CSS + JavaScript，无框架依赖
- 所有数据为模拟数据，关键操作（打卡、库存修正、家属添加）通过 localStorage 持久化
- 购药下单后库存自动更新，患者和家属页面同步刷新
- 移动端优先设计，max-width 430px 居中展示
- 浏览器直接打开即可运行
